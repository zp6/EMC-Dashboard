# Bug Bounty Program - EMC Dashboard

## Scope
EdgeMatrix Chain (EMC) Dashboard application and API.

## Rewards
| Severity | Reward |
|----------|--------|
| Critical | $1,000 - $5,000 |
| High | $500 - $1,000 |
| Medium | $100 - $500 |
| Low | $50 - $100 |

## In Scope
- XSS vulnerabilities
- Authentication bypasses
- API injection issues
- Data exposure bugs
- Node management vulnerabilities
- Dashboard access control issues

## Reporting
1. Email: security@edgematrix.pro
2. Include detailed steps to reproduce
3. Rate the severity
4. Do not publicly disclose

## Response
- 48h acknowledgment
- 7-30 days fix timeline based on severity
